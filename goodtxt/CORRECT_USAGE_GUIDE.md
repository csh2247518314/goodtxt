# 🚀 GoodTxt 系统启动指南

## ❌ 错误方法（不要使用）：
python3 quick_start.py      # ❌ 这个文件不存在
python3 launcher.py         # ❌ 这个文件不存在  
python3 env_checker.py      # ❌ 这个文件不存在

## ✅ 正确方法：

### 1. 首先进入项目目录
cd /path/to/your/goodtxt

### 2. 使用超级启动器
python3 super_launcher.py

### 3. 在菜单中选择选项
- 选择 [1] 完整部署（推荐）
- 选择 [2] 快速启动
- 选择 [3] 环境检查

## 🔧 如果您之前下载了旧版本：

### 更新到最新版本：
```bash
# 拉取最新代码
git pull origin main

# 或者重新克隆
rm -rf /root/goodtxt
git clone https://github.com/csh2247518314/goodtxt.git
cd goodtxt
```

### 安装Docker（如果需要）：
```bash
# 运行我们的国内镜像安装脚本
./install.sh
```

## 🎯 快速启动：
```bash
cd /root/goodtxt  # 进入项目目录
python3 super_launcher.py  # 使用超级启动器
```

然后按照提示操作即可！
