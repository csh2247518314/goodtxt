## 🎯 **立即开始使用**

### **1. 启动系统**
```bash
# 进入项目目录
cd /workspace/goodtxt

# 使用智能启动器
python3 super_launcher.py
```

### **2. 访问系统**
- 🌐 **前端界面**: http://localhost:3002
- 📡 **后端API**: http://localhost:8000  
- 📖 **API文档**: http://localhost:8000/docs

### **3. 用户登录**
- 👤 **默认账户**: admin
- 🔑 **默认密码**: admin123456
- ➕ **注册新账户**: 点击"立即注册"

## 🎨 **核心功能使用**

### **创建小说项目**
1. 登录后点击"新建项目"
2. 填写小说基本信息：
   - 📝 标题
   - 📚 类型（科幻、言情、悬疑等）
   - 📏 长度（短篇、中篇、长篇）
   - 🎯 主题
   - 👥 目标读者

### **开始AI生成**
1. 选择创建的项目
2. 点击"开始生成"
3. 系统会自动生成章节内容
4. 实时查看生成进度

### **质量监控**
1. 查看生成章节的质量评分
2. 可以重新生成不满意的章节
3. 编辑和优化内容

## ⚙️ **高级配置（可选）**

### **配置AI API密钥**
如果需要更强的AI功能，可以配置API密钥：

```bash
# 1. 复制环境模板
cp .env.template .env

# 2. 编辑配置文件
nano .env

# 3. 添加您的API密钥
AI_DEEPSEEK_API_KEY=your_deepseek_key
AI_QWEN_API_KEY=your_qwen_key
AI_MINIMAX_API_KEY=your_minimax_key
```
